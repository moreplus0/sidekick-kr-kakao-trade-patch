Sidekick KR Kakao Trade Patch v2026.6.5

POE/POE2 거래 보조 프로그램 공식 Sidekick v2026.6.5 portable에 한국 POE/POE2 거래소 Kakao Games 주소 패치를 적용한 비공식 임시 배포본입니다.

공식 Sidekick v2026.6.5는 2026-06-19 이후 바뀐 trade API 응답 구조는 수정했지만, 한국어 거래소 기본 주소는 아직 poe.game.daum.net으로 남아 있습니다. 그래서 공식 업데이트 후에도 한국어 POE2 거래 검색에서 API 오류가 반복될 수 있어, 공식 6.5 기반에 Kakao 주소 패치만 추가한 버전을 다시 만들었습니다.

ZIP 다운로드 릴리스 페이지 GitHub 저장소

문제 증상

한국어 POE2 환경에서 Sidekick 가격 검색을 실행하면 아래처럼 API 통신 실패 화면이 나올 수 있습니다.

또는 거래 API 인증 관련 안내가 표시되면서 검색 결과가 나오지 않을 수 있습니다.

2026-06-19 전후에는 매물 목록을 가져오는 단계에서 아래처럼 fetch 오류도 발생했습니다. 이 응답 구조 문제는 공식 Sidekick v2026.6.5에 반영됐고, 이번 배포본도 공식 6.5를 기반으로 합니다.

  • Sidekick failed to communicate with the API.
  • Sidekick failed to communicate with the trade API.
  • The trade website requires authentication, which Sidekick does not support currently.
  • Sidekick could not fetch the listings from the trade API.
  • 검색 화면 우측에 No results found가 같이 표시됨
  • Sidekick을 삭제 후 재설치하거나 캐시를 초기화해도 한국어 가격 검색 오류가 반복됨

이번 버전에서 바꾼 부분

공식 Sidekick v2026.6.5 포터블을 기반으로 두고, 한국어 게임 언어 설정 파일인 GameLanguageKo.cs의 거래소 주소 4개만 Kakao Games 주소로 바꿨습니다. 

기존:
https://poe.game.daum.net/trade/
https://poe.game.daum.net/api/trade/
https://poe.game.daum.net/trade2/
https://poe.game.daum.net/api/trade2/

변경:
https://poe.kakaogames.com/trade/
https://poe.kakaogames.com/api/trade/
https://poe.kakaogames.com/trade2/
https://poe.kakaogames.com/api/trade2/

이번 ZIP에서 직접 교체된 파일은 current\Sidekick.Data.dll입니다. 공식 6.5의 API 응답 구조 수정 DLL들은 그대로 유지했습니다.

한국어 검증

Kakao POE2 거래 API에서 한국어 아이템명과 한국어 옵션이 정상 반환되는 것을 확인했습니다. 

OfficialSidekick=Sidekick v2026.6.5
Build=dotnet build Sidekick.Data.csproj -c Release
ContainsKakaoUrl=True
ContainsDaumUrl=False
KakaoFetchStatus=200
ItemName=경계하는 시야
BaseType=에메랄드 반지
ExplicitModDescription=[Accuracy|정확도] +143

사용 방법

  1. ZIP 파일을 원하는 폴더에 압축 해제합니다.
  2. 압축을 푼 폴더의 Sidekick.exe를 실행합니다.
  3. Sidekick 설정에서 게임 언어와 UI 언어를 Korean으로 맞춥니다.
  4. 리그를 현재 POE2 리그로 선택합니다.
  5. 영어 강제 검색 옵션은 꺼두는 것을 권장합니다.
  6. POE2에서 아이템 위에 마우스를 올리고 Sidekick 가격 검색 단축키를 사용합니다.
